¿Qué es FileVault?

FileVault es una aplicación web para compartir archivos de forma segura mediante tokens únicos. Sin registro, sin cuentas, sin complicaciones.

Sube tu archivo, obtén un token generado automáticamente y compártelo con quien quieras. Solo quien tenga el token puede descargar el archivo.

Límites y restricciones
↑
Subida: máx. 500 MB Por archivo. Sin límite de archivos totales.
↓
Descarga: sin límite Un token puede usarse para descargar el mismo archivo las veces que quieras.
⏱
Duración: sesión activa Los archivos existen mientras la pestaña esté abierta (demo local).
Formato del token
A3F7
—
K9P2
—
X1M8

El token se genera automáticamente al subir el archivo. Es único, aleatorio y no puede adivinarse. Tiene 3 segmentos de 4 caracteres (letras y números).
